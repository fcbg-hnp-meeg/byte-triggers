**Byte triggers**
=================

This package offers simple objects to deliver triggers on serial port, parallel port or
LSL streams.

Install
-------

.. tab-set::

    .. tab-item:: PyPI

        .. code-block:: console

            $ pip install byte-triggers

    .. tab-item:: Source

        .. code-block:: console

            $ pip install git+https://github.com/fcbg-platforms/byte-triggers
